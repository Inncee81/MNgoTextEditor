• Developed a personalized Drive Application like Google Drive, that can be used for storing files and folders on my server.
  • Used AJAX to communicate with API built on PHP.
  • Used MySQL database to store user data after encrypting it.
  • Created a custom context menu in JavaScript to replace the default one and added user control options (like open, rename, delete, share, details, etc) to that.
  • Technologies: JavaScript, jQuery, AJAX, MySQL, PHP, Bootstrap
  • GitHub Repository: https://github.com/adityasuman2025/MNgoDrive
  • Web link: drive.mngo.in

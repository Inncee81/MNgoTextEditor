• A Web Application for IIT Patna where students can submit feedback of courses taught by profs.
  • This has been hosted on the college website and the final year students have filled the feedback using this module. The module reported no bugs and students have responded positively
  • Technologies: PHP, MySQL, JavaScript, AJAX
  • Web link: https://www.iitp.ac.in/feedbackIITP/
